Project Overview

The Option Pricing Model project involves developing and analyzing various models to determine the fair price of European call options. Utilizing Python and libraries such as `yfinance`, `pandas`, `numpy`, `scipy`, `matplotlib`, and `seaborn`, this project implements the Black-Scholes model, Binomial Tree model, and Monte Carlo simulations to price options on Apple Inc. (AAPL) stock.

The project includes fetching real-time stock and option data, calculating implied and historical volatilities, and visualizing the results through comprehensive graphs.

Key Code Snippet


# Black-Scholes Call Option Pricing Function
def black_scholes_call(S, K, T, r, sigma):
    if T <= 0:
        return max(S - K, 0)  # Intrinsic value
    d1 = (np.log(S / K) + (r + 0.5 * sigma ** 2) * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    call_price = S * norm.cdf(d1) - K * np.exp(-r * T) * norm.cdf(d2)
    return call_price
