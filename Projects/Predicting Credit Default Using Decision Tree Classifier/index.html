Project Overview

This project focuses on credit risk analysis using the German Credit Dataset from OpenML. The analysis involves building a Decision Tree Classifier to predict credit defaults. The model is trained on features like personal and financial details of customers, with metrics such as accuracy, classification reports, and confusion matrices being used to evaluate performance.

Project Details

  • Dataset: German Credit Data from OpenML
  • Model Used: Decision Tree Classifier
  • Technologies Used: Python, pandas, numpy, scikit-learn, matplotlib, seaborn, graphviz
  • Date: June 2024

Conclusions

The Decision Tree Classifier proved effective in predicting credit defaults for the German Credit dataset. By visualizing feature importance, we were able to identify key factors that contribute to credit risk. The model achieved reasonable accuracy, and cross-validation was performed to ensure robustness. Hyperparameter tuning was also used to further optimize the model.
